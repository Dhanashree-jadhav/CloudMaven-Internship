# 📊 ELK Stack Logging Dashboard

## 📌 Project Overview

This project demonstrates a basic logging and monitoring setup using the **ELK Stack (Elasticsearch, Logstash, Kibana)**. Logs are collected using **Filebeat**, stored in Elasticsearch, and visualized using Kibana dashboards.

---

## 🛠️ Tech Stack

* Elasticsearch
* Logstash
* Kibana
* Filebeat
* Docker & Docker Compose

---

## 🚀 Features

* Real-time log collection and monitoring
* Visualization of logs using Kibana
* Interactive dashboard with multiple panels
* Filtering logs based on agent and dataset

---

## 📂 Dashboard Components

### 1. Logs Over Time

* Displays log count over time
* Helps identify traffic trends and spikes

### 2. Logs by Agent

* Shows logs generated by different agents (e.g., Filebeat)

### 3. Logs by Dataset

* Categorizes logs based on dataset (e.g., system, application)

### 4. Total Logs

* Displays total number of logs in a selected time range

### 5. Filters

* Dynamic filtering using fields like:

  * `agent.type`
  * `data_stream.dataset`

## ⚙️ Setup Instructions

### 1. Clone the Repository

```bash
git clone https://github.com/Dhanashree-jadhav/ELK-PROJECT.git
cd ELK-PROJECT 
```

### 2. Start ELK Stack using Docker

```bash
docker-compose up -d
```

### 3. Access Kibana

Open browser:

```
http://localhost:5601
```
